TFE功能

安装Node：https://nodejs.org/en/ 安装NPM包管理器：https://www.npmjs.com/ 下载TFE套件： npm install tfe -g 项目初始化：tfe init 创建模块：tfe add c|p * 代码规范检查：tfe lint check 代码格式化：tfe lint format 创建服务器：tfe dev
